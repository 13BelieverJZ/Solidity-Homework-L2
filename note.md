# 数字签名
    以太坊使用的数字签名算法叫：双椭圆曲线数字签名算法（ECDSA），基于双椭圆曲线“私钥-公钥”对的数字签名算法
    作用：
    1.身份认证：证明签名方是私钥的持有人
    2.不可否认：发送方不能否认发送过这个消息
    3.完整性：通过验证针对传输消息生成的数字签名，可以验证消息是否在传输过程中被篡改

# ECDSA标准
    1.签名者：私钥对消息创建签名
    2.其他人：使用消息和签名恢复签名者的公钥并验证签名
